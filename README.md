## Stellar Game Framework

Stellar is a simple java 2D game framework. 
Designed to be very easy to use.

### Features

- [x] 2D Render Engine
    - [x] Camera
    - [ ] Lighting
    - [ ] Custom Shader
- [x] User Input
    - [x] Keyboard
    - [ ] Mouse
    - [ ] Joystick
- [x] Collision Detection
    - [ ] AABB
    - [x] SAT
    - [ ] Circle-Polygon
- [ ] Sound Support
