## Stellar Game Framework

Stellar is a simple java 2D game framework. 
Designed to be very easy to use.

### Features

- [x] 2D Render
    - [x] Line
    - [x] Circle
    - [x] Sprite
    - [x] Camera
    - [x] Postprocessing
    - [ ] Particles
    - [ ] Lighting
- [x] User Input
    - [x] Keyboard
    - [ ] Mouse
    - [ ] Joystick
- [x] Collision Detection
    - [x] Polygon-Polygon
    - [x] Circle-Polygon
    - [x] Circle-Circle
- [ ] Sound Support
